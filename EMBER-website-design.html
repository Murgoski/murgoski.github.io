About the Project

EMBER is a fictional restaurant website concept created entirely by me — from the name, brand idea, and overall visual identity to the complete website design. The goal was to design a modern and elegant web experience that combines warm tones, minimalist typography, and smooth visual flow to evoke the feeling of a premium dining atmosphere. The project includes desktop and mobile versions, carefully adapted for responsive behavior and optimal usability across devices.

Things I Have Learnt

Working on this project helped me grow creatively and technically. It was also my first website design made with the open-source tool Penpot, that helped me understand the workflow between UI design and front-end development.


Through this project, I learned to:

  • Create consistent visual systems and reusable design components.
  • Build responsive layouts for both desktop and mobile.
  • Apply principles of alignment, spacing, and color contrast.
  • Focus on usability and user flow.
  • Better understand how fonts and color choices shape the overall look and feel.
