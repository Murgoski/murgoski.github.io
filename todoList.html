    To-do List - Main page

    The image showcases a "To-Do List" interface.

    The interface is divided into two main columns:

    • New Task Column (Left side):
      • Contains a button labeled "Create a Task" which suggests users can add new tasks by clicking it.
    • My Tasks Column (Right side):
      • Lists several tasks along with their due dates and categories.
      • In the top right corner of this column, there are three additional buttons:
        • Search button - Allows users to search the tasks by task name or task label.
        • Feedback button - Enables users to send feedback to the admin for upgrades and improvements.
        • Trashcan button - Enables users to delete all tasks at once from the list.

    New Task Form

    The highlighted part of the image shows the "New Task" form.

    This form is used to create a new task and includes the following fields:

    • Task Name - A text field where users can enter the name of the task.
    • Select Task Label - A dropdown menu to choose a label for the task.
    • Start Date - A date picker to select the start date for the task.
    • Due Date - A date picker to select the due date for the task.
    • Add a Description - A text area where users can provide additional details about the task.
    • Add a Task Button - A button labeled "Add a task" for submitting the new task to the list.

    Task content

    Each task is in its own element to ensure clarity and transparency.

    Every task comprises the following components:

    • Checkbox button - Allows users to mark tasks as completed when checked
    • Task name - Displays the name of the task.
    • Due date - Indicates the deadline or due date for completing the task.
    • Buttons:
      • Clipboard button - Displays additional information for the task.
      • Trashcan button - Enables users to delete individual task from the list.
    • Task label - Provides categorization for the task. Each label is assigned a color.

    Task additional information

    The Clipboard button, when clicked, opens a popup window with additional information about a certain task. The task description in the individual task closely resembles the information provided in the task description popup.
    The main difference lies in the additional details featured in the popup, such as the start date and a more detailed description, which enriches the understanding of the task's specifics.

    Feedback form

    The form pops up when the user clicks on the Feedback button.

    It enables users to send feedback to the admin for upgrades and improvements.

    The form features three fields:

    • Full Name
    • Email Address
    • Feedback Textarea

    This feature helps in gathering user suggestions and identifying areas for enhancement.
